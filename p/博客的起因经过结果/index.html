博客的起因经过结果

从第一台阿里云服务器到把域名接到 GitHub Pages，这是我开始折腾博客的全过程。

2026-01-18（更新于2026-03-01）

纯粹是闲出屁来了，最开始是因为在b站上看到有推销服务器的，恰巧我当时对这个比较感兴趣。于是我就在想我是不是也可以整一台云服务器，当机立断就找妈妈要了100块钱，然后就在网上选到了阿里云的2h2g加峰值200m宽带，还记得当时阿里云有个活动，是花38元抢一年的服务器，很幸运我当时抢到了，然后我就在想，光有服务器没域名也不行啊，也是阿里云新人活动，直接花1元巨款购入一年的whylove.top很棒啊很棒啊，那么这个时候是不是就可以干点什么了呢，恰巧我之前闲着无聊在GitHub page上托管了一个博客，因为GitHub默认给的域名太丑了也不彰显个性，于是我就想把whylove这个域名反代到github给的域名上，最终历经了重重磨难后，我成了。
