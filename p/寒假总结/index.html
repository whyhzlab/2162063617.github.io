重点 2：博客回国 + CDN 加速

既然有了合规域名那肯定是要干点什么的吧，就比如说是博客，没错，就是现在你看到的东西。最先博客是托管在国外 github 平台的，然后由于政策原因在国内打开很不稳定，现在备案之后可以使用国内 CDN 加速，下面是效果图 可以看到非常逆天的加载速度

重点 3：服务器项目落地

然后关于服务器目前我在上面部署了两个项目，一个 qqbot 和一个网盘。qqbot 是用开源项目 astrbot，目前只给了我同桌玩玩，不过这 ai 角色扮演真这么好玩？我同桌几天给我刷了几万条消息，也是个逆天，我对话了几次感觉没啥意思 网盘也是用的开源项目 openlist，你目前看到的所有图片应该都存在这个网盘上博客现在已经不再使用外链图片，改为使用本地图片
